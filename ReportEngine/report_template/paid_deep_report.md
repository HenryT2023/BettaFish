### **付费深度研究报告**模板

- **1.0 执行摘要**
  - 1.1 核心发现（300字，一页纸概括全文）
  - 1.2 关键数据指标
- **2.0 数据全景**
  - 2.1 多平台数据汇总（海外 + 国内）
  - 2.2 数据来源与采集方法说明
  - 2.3 时间范围与样本量
- **3.0 深度分析**
  - 3.1 趋势分析（数据图表 + 文字解读）
  - 3.2 竞品对比（国内外主要玩家）
  - 3.3 机会识别（蓝海 / 差异化切入点）
  - 3.4 风险评估（政策 / 市场 / 技术风险）
- **4.0 情感分析**
  - 4.1 舆论风向量化（正面/中性/负面占比）
  - 4.2 关键意见领袖（KOL）观点汇总
  - 4.3 消费者/用户真实反馈摘录
- **5.0 行动建议**
  - 5.1 卖家/品牌方建议
  - 5.2 投资者/创业者建议
  - 5.3 短期（1-3月）vs 中期（3-12月）策略
- **6.0 数据附录**
  - 6.1 原始数据表格
  - 6.2 参考来源链接
  - 6.3 术语解释
